## 🎯 How It Works

### The Training Flow

1. **Welcome Menu** → User sees conversation starters
2. **Select Scenario** → Choose difficulty and topic
3. **Blind Conversation** → ISA doesn't see client profile
4. **Natural Dialogue** → ISA must ask questions to discover needs
5. **Appointment Goal** → Focus on setting a meeting
6. **Profile Reveal** → After appointment, client details are shown
7. **Feedback & Score** → Get 0-100 score with detailed analysis

### Key Features

- **Hidden Profiles**: Client personality and details are secret
- **DISC Assessment**: Each client has unique personality type
- **Real Scenarios**: Diverse situations (buyers, investors, sellers, etc.)
- **Performance Scoring**: Evaluates questioning, listening, adaptation
- **Comprehensive Feedback**: Strengths, improvements, metrics
- **Responsive Design**: Works on desktop, tablet, mobile

## 🚀 Getting Started

### 1. Backend Setup (2 minutes)
```bash
cd backend
npm install
npm start
# Server runs on http://localhost:5000
```

### 2. Frontend Setup (2 minutes)
```bash
cd frontend
npm install
npm start
# App opens at http://localhost:3000
```

### 3. Start Training
- Click any conversation starter
- Select a scenario
- Engage in conversation without seeing client profile
- Try to set an appointment
- Get your score and feedback

**Total time to running**: ~5 minutes
