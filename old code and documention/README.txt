   (You'll see: "backend running on port 5000")

3. START FRONTEND (Terminal 2):
   cd frontend
   npm install
   npm start
   (Browser will open to http://localhost:3000)

4. BEGIN TRAINING:
   - Click any conversation starter
   - Select a scenario
   - Have a conversation
   - Set an appointment
   - Get your score!
